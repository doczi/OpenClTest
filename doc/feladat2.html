Feladat:

Készíts alkalmazást ami a rendszer OpenCL API-ról gyűjt információt, amit JSON formában megjelenít a konzolon. Elvárások az alkalmazással kapcsolatban:

  • Az alkalmazás egy library-t használjon a szükséges információ összegyűjtésére.
  • A library biztosítson osztályokat, a lekért információ tárolására.
  • Az adattároló osztályok legyenek függetlenek a konkrét szerializációtól (Lehessen később lecserélni a szerializációt pl JSON-ről XML-re).
  • Az OpenCL kontextet a library hozza létre. Az összes elérhető device-ról, kérdezzen le minél több információt.
  • Az OpenCL library-t dinamikusan töltse be (LoadLibrary, dlopen), NE linkeljen hozzá. Az API szükséges függvényeit futásidőben kérje le.
    • A betöltésre külső könyvtár használható, de nem követelmény. pl: https://code.google.com/p/clew/
  • Az alkalmazásnak lehessen megadni az OpenCL library elérési útját.
  • Az alkalmazás próbáljon meg lefordítani egy egyszerű kernelt. A fordított kernel kódja, és a fordítás státuszát tartalmazza a kimenet.

Platform:

Tetszőlegesen választható az alábbiak közül: Windows, Linux, MacOS

A kóddal kapcsolatos elvárások:

  • Az implementáció készüljön C++ nyelven (c++11 használható)
  • Legyen átgondolt, objektum orientált, újrafelhasználható, jól struktúrált
  • A kód főbb strukturális elemei legyenek dokumentálva
  • A kód legyen tesztelve. A lehetséges hibaesetekre ki kell térni, legalább kódban megjegyzés szinten.

A kódhoz legyen mellékelve leírás, ami tartalmazza hogyan kell projectet lefordítani, elindítani, milyen függőségei vannak, milyen platformon lett tesztelve. A csomag tartalmazzon egy projectfájlt, amivel a program fordítható, és kipróbálható.
